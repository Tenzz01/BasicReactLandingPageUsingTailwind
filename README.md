
# React + Tailwind CSS Landing Page

## Overview
A modern, responsive landing page built with React and Tailwind CSS.

## Features
- Responsive design
- Clean, modern UI
- Fast performance
- Easy to customize

## Tech Stack
- **React** - UI library
- **Tailwind CSS** - Utility-first CSS framework
- **Vite** (or Create React App) - Build tool

## Getting Started

### Prerequisites
- Node.js (v14+)
- npm or yarn

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

## Project Structure
```
src/
├── components/
│   ├── Header.jsx
│   ├── Hero.jsx
│   ├── Features.jsx
│   └── Footer.jsx
├── App.jsx
└── index.css
```

## Customization
Modify Tailwind classes in your components to customize colors, spacing, and layout.
